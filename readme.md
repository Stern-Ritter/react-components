## Созданы простые React компоненты:

1. заголовок (с поддержкой уровней);
2. параграф текста;
3. пробельный блок (с горизонтальной линией);
4. схлопывающийся блок (может сворачиваться в заголовк или показывать контент);
5. картинка.
